# Política de Privacidade - FieldCheck-in

Este repositório armazena e hospeda a Política de Privacidade oficial do aplicativo **FieldCheck-in**.

O FieldCheck-in é um aplicativo de uso estritamente corporativo e restrito (B2B), destinado ao controle operacional de registros de entrada (check-in) e saída (check-out) com validação por imagem para empresas previamente autorizadas.
